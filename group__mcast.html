m1d m1d_cast_v1d ( v1d  a)
related

Casts a v1d vector into a m1d vector-mask.

May be faster than m1d_cvt_v1d(v1d a) if a can act as a mask (ie: every elements are either 0 or 0xFFFFFFFFFFFFFFFF). Undefined behaviors otherwise.

If a can act as a mask, the following statements are equivalent for every vector v1d vec:

  • v1d_and(a, vec)
  • v1d_mask_move(m1d_cast_v1d(a), vec)
Operation
for j from 0 to 0
i := j*64
if dst.bits[i:i+63] == 0xFFFFFFFFFFFFFFFF
mask[j] := 1
else if dst.bits[i:i+63] == 0
mask[j] := 0
endif
endfor
