Detailed Description

Type for the manipulation of 4 float packed into one single vector. This structure does not actually exist. Depending on the target architecture, it can be:

  • A preprocessor alias
  • A typedef
  • A struct defined in another way
  • A union

Sizes of this vector:

  • 4 elements of type float (32 bits)
  • 16 bytes
  • 128 bits
Attention
This definition does not correspond to any actual implementation and should never used directly.
The actual implementation depends on the target architecture.
Use PintS functions instead.
