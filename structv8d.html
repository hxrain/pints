Detailed Description

Type for the manipulation of 8 double packed into one single vector. This structure does not actually exist. Depending on the target architecture, it can be:

  • A preprocessor alias
  • A typedef
  • A struct defined in another way
  • A union

Sizes of this vector:

  • 8 elements of type double (64 bits)
  • 64 bytes
  • 512 bits
