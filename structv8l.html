Detailed Description

Type for the manipulation of 8 int64_t packed into one single vector. This structure does not actually exist. Depending on the target architecture, it can be:

  • A preprocessor alias
  • A typedef
  • A struct defined in another way
  • A union

Sizes of this vector:

  • 8 elements of type int64_t (64 bits)
  • 64 bytes
  • 512 bits
