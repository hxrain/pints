Detailed Description

Type for the manipulation of 4 double packed into one single vector. This structure does not actually exist. Depending on the target architecture, it can be:

  • A preprocessor alias
  • A typedef
  • A struct defined in another way
  • A union

Sizes of this vector:

  • 4 elements of type double (64 bits)
  • 32 bytes
  • 256 bits
Attention
This definition does not correspond to any actual implementation and should never used directly.
The actual implementation depends on the target architecture.
